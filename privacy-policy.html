한국어 버전

1. 목적

본 개인정보처리방침은 모바일 애플리케이션 이른둥이 (Preemie / プリミー) (이하 “앱”)이 이용자 정보 처리와 관련하여 이용자에게 필요한 사항을 알리기 위해 작성되었습니다.

2. 중요 안내 (앱의 데이터 처리 방식)

핵심 요약: 이 앱은 서버를 사용하지 않으며, 모든 사용자 데이터는 기기 내 Hive 데이터베이스에 로컬 저장됩니다. 앱은 이용자의 개인정보를 수집하거나 외부로 전송·공유하지 않습니다.

3. 저장되는 정보(로컬)

  • 앱 설정(예: 언어, 알림 설정)
  • 사용자가 앱 내 입력한 값(예: 메모, 설정값 등)
  • 오프라인 사용 기록(기능 사용 시각 등)이 로컬에 저장될 수 있음

4. 서버 전송 / 제3자 제공

앱은 외부 서버에 데이터를 전송하지 않으며, 광고·분석 SDK 및 제3자 제공을 하지 않습니다.

5. 결제 및 인앱구매(해당 시)

인앱 결제가 있을 경우 결제는 Google Play 등 스토어 결제 시스템을 통해 처리되며, 개발자는 카드번호 등의 민감정보를 취급하지 않습니다. (결제가 없다면 본 항목은 적용되지 않습니다.)

6. 백업 및 기기 클라우드 저장

기기 또는 OS의 자동 백업을 사용하면 앱의 로컬 데이터가 클라우드 백업에 포함될 수 있습니다. 이를 원치 않으면 기기 설정에서 앱 데이터 백업을 비활성화해 주세요.

7. 데이터 보유 및 삭제

데이터는 사용자 기기에 보관됩니다. 앱 삭제 또는 앱 내 삭제 기능을 사용하면 기기 내 데이터가 삭제됩니다. 단, 클라우드 백업에 포함된 데이터는 백업 제공자 정책에 따라 별도 삭제가 필요할 수 있습니다.

8. 이용자의 권리

서버 보관이 없으므로 서버 기반 열람·정정 요청은 해당되지 않습니다. 기기 내 데이터의 열람·수정·삭제는 앱 내 기능을 이용하십시오. 추가 문의는 아래 연락처로 요청해 주세요.

9. 아동(미성년자)

아동이 사용하는 경우 보호자 동의가 필요한 기능이 있다면 앱 내에서 안내합니다. 본 앱은 개인 식별 정보의 외부 전송을 수행하지 않습니다.

10. 변경 및 고지

정책은 앱 기능 또는 관련 법령 변경에 따라 수정될 수 있으며, 중요한 변경은 앱 내 공지 또는 GitHub 페이지를 통해 안내합니다.

1. Purpose

This Privacy Policy explains how the mobile application Preemie (이른둥이 / プリミー) (the “App”) handles user information and informs users of their rights.

2. Important Notice (Data Handling)

Summary: The App uses no external servers. All user data is stored locally on the device via Hive. The App does not collect, transmit, or share personal data to third parties.

3. Information Stored (Local)

  • App settings (e.g., language, notifications)
  • User-entered values within the App (e.g., notes, local preferences)
  • Offline usage records (e.g., timestamps) stored locally

4. No External Transmission / Third-Party Sharing

No data is sent to external servers, and no analytics/ads SDKs are used.

5. Payments & In-App Purchases (If Any)

If provided, purchases are processed by the app store’s payment system (e.g., Google Play). The developer does not handle sensitive payment data. (If not offered, this section does not apply.)

6. Backups & Cloud

OS/device backups may include the App’s local data. To avoid cloud backups, disable app data backup in device settings.

7. Data Retention & Deletion

Data remains on the user’s device. Deleting the App or using in-App delete features removes local data. Copies in cloud backups may require separate deletion via the backup provider.

8. Your Rights

Server-side access/correction requests are not applicable. Users can view/edit/delete local data via in-App functions. For assistance, contact us below.

9. Children

If children use the App, parental consent is required for any features that would collect personal data. The App itself does not transmit personal data externally.

10. Changes

We may update this policy due to feature or legal changes; significant updates will be announced in-App or on the project’s GitHub page.

1. 目的

本プライバシーポリシーは、モバイルアプリプリミー（Preemie／이른둥이）（以下「本アプリ」）におけるユーザー情報の取扱いについて説明するものです。

2. 重要なお知らせ（データ処理）

要約：本アプリは外部サーバーを使用せず、すべてのユーザーデータは端末内の Hive にローカル保存されます。個人情報の収集・送信・第三者提供は行いません。

3. 端末内に保存される情報

  • アプリ設定（言語、通知設定 など）
  • ユーザーがアプリ内で入力した値（メモ、各種設定 など）
  • オフラインの利用記録（機能の利用時刻 など）

4. 外部送信／第三者提供

データの外部送信は行わず、広告・解析のための第三者SDKも使用しません。

5. 決済・アプリ内課金（該当する場合）

課金がある場合、決済は各アプリストアの仕組みにより処理され、開発者がカード情報等の機微情報を直接取り扱うことはありません。（課金がなければ本項は該当しません。）

6. バックアップ・クラウド

端末／OSのバックアップ設定により、アプリのローカルデータがクラウドに含まれる可能性があります。クラウドへのバックアップを望まない場合、端末設定で当該バックアップを無効にしてください。

7. データの保持・削除

データはユーザー端末に保持されます。アプリの削除またはアプリ内の削除機能により端末内データは削除されます。クラウドバックアップに保存されたコピーは、バックアップ提供者側での削除が必要となる場合があります。

8. 利用者の権利

サーバー保管がないため、サーバー側での開示・訂正請求は対象外です。端末内データの確認・編集・削除はアプリ内機能で行ってください。サポートが必要な場合は、下記窓口までご連絡ください。

9. 児童（未成年）

児童が本アプリを利用する場合、個人情報の収集が必要となる機能については保護者の同意を得てください。ただし本アプリ自体は個人情報の外部送信を行いません。

10. 改定

本ポリシーは、機能変更や法令の変更に応じて更新されることがあります。重要な変更はアプリ内通知またはGitHubページで告知いたします。
